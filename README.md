## 사용법

프로젝트 최상위 폴더에
school.json 과 student.json 파일을 생성함미다  
school.json에 다음과 같은 형식으로 내용을 작성함미다

```json
{
  "name": "학교 이름 (정확해야됨)",
  "level": "학교급 (예시: 중학교)",
  "city": "학교가 있는 곳 (예시: 대구)"
}
```

student.json에는 다음과 같은 형식으로 내용을 작성함미다

```json
{
  "name": "자신의 이름",
  "birth": "생년월일 (YYMMDD)",
  "pw": "자가진단 비밀번호"
}
```

이제 프로젝트를 빌드함미다

```sh
$ yarn build
```

그리고 실행합니다

```sh
$ yarn start
```

끝!
